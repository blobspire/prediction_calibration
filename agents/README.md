# Agents

Project governance and agent-facing planning files live here.

- `AGENTS.md`: agent operating contract, scientific guardrails, workflow rules, and review focus.
- `PROJECT_BRIEF.md`: research objective, scope, methodology requirements, and definition of done.
- `ROADMAP.md`: phase-gated implementation roadmap.
- `TASK_LOG.md`: chronological implementation ledger and current follow-up tasks.
- `CURRENT_CAPABILITIES.md`: pointer to the canonical capability registry in `docs/CURRENT_CAPABILITIES.md`.

The current user-facing operating manual remains the root `README.md`.
Detailed methodology and audit notes remain under `docs/`.
